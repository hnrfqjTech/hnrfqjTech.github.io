opAdmin-boilerplate

基于 Vue-Cli3 的运营管理后台脚手架

快速上手 →

快速开发

参照指引文档和示例代码，可以帮助您快速的实现业务功能。

Vue 驱动

享受 Vue + webpack 的开发体验。

特性支持

样式编译器SASS、组件化、ES2015、请求优化、插件、vuex-helper、可视化配置页面 等。

# 像数 1, 2, 3 一样容易

# 安装
npm i -g @hnrfqj/rfqj-cli

# 使用
rfqj clone opAdmin
cd opAdmin
npm i
npm run dev

TIP

请确保你的 Node.js 版本 >= 6
