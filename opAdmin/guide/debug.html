# 快速上手

  • 查看基于 vue-cli3 创建的应用状态：vue ui
  • 查看 webpack 配置项：vue inspect
  • 调试工具：浏览器插件 Vue.js devtools，可方便查看页面结构、组件、事件、Vuex 等
  • 接口代理转发：vue.config.js 中 devServer.proxy 配置
