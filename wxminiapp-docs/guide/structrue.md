# 目录结构

```
├── build              // 构建
├── config             // 构建配置
├── dist               // 小程序代码
│   ├── common
│   ├── components
│   ├── pages
│   │   ├── home
│   │   ├── pkg
│   │   └── user
│   └── static
│       ├── assets
│       └── weui
├── src               // 源码
│   ├── api           // 接口
│   ├── components    // 组件
│   ├── config        // 配置文件
│   ├── css           // 样式
│   ├── extend        // Vue 拓展
│   │   └── plugin    // Vue 插件
│   ├── pages         // 页面
│   │   ├── home
│   │   ├── pkg       // 分包
│   │   └── user
│   ├── store         // 数据
│   └── utils         // 工具函数
└── static            // 静态资源
    ├── assets
    └── weui
```
